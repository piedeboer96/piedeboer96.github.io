Internship at Rabobank

Sep 2023 - Feb 2024

For approximatly six months I continued work on an existing Rabobank project. I was challenged with the task of building improvements to the existing quantum approach.

Quantum computing can potentially solve complex optimization problems faster than classical computers by leveraging quantum superposition and entanglement to explore multiple solutions simultaneously.

With updated and improved solutions, I was able to provide a solution to Rabobank although due to hardware limitations still a bit noisy, but with drastic improved complexity.

Portfolio Optimization using Quantum Methods - Contributed to an ongoing Rabobank project by further developing their quantum-based approach for (non-linear) portfolio optimization using the D-Wave package and Python. Leveraged the D-Wave API to run the optimization algorithms on real quantum hardware. This project had an interesting focus on investing in a sustainable manner, e.g., as listed in living up to certain CO2 emission.
