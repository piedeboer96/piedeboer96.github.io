Conditional Diffusion Models for ECG Signal Denoising üè•

Oct 2023 - Jun 2024

Developed a novel state-of-the-art approach to ECG signal denoising by leveraging an image upscaling model (SR3), a conditional diffusion model. By transforming ECG signals into Gram Angular Fields, we were able to apply image-based techniques. We aimed to have one model that was able to remove multiple types of noise, where traditional methods are mostly focused on removing a specific type of noise. Our research demonstrated superior performance compared to traditional signal processing methods, such as adaptive filters, low-pass filters, and moving averages. The models were trained from scratch using Google Cloud using just the trial period and a low-tier GPU.

Packing Problem üì¶

Sep 2021 - Jan 2022

Developed algorithms from scratch to pack 2D or 3D packages efficiently into a container. Initially built a Tetris-style solver, gradually advancing to 3D packages. This project was a great work in the domain of algorithms, especially as it serves as an excellent sandbox for learning how to work with greedy and genetic algorithms. Additionally, it provided an opportunity to implement Donald Knuth's "Dancing Links," which can also be used for the famous exact cover problem, Sudoku. This project included GUIs programmed in Swing and JavaFX.
